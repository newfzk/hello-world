# Git学习笔记

## 初始化 

1. 设置用户名

    `git config --global user.name "用户名"`

2. 设置邮箱

    `git config --global user.email "邮箱"`

3. 全局存储账号

    `git config --global credential.helper store`

4. 建立git管理文件

    `git init`

## 添加文件 (add)

1. 查看版本库状态

    `git status`

2. 添加文件

将没有放入版本库的文件（unstaged状态）添加进版本库（staged）
    `git add 文件名`
